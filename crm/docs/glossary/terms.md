# Глосарій Shveyka MES

## Бізнес-терміни

| Термін | Опис |
|--------|------|
| **Виробниче замовлення** (Production Order) | Верхньорівневий документ виробництва. Створюється менеджером, проходить етапи: draft → approved → launched → completed → warehouse_transferred → closed |
| **Позиція замовлення** (Order Line) | Конкретна модель та кількість у замовленні. Замовлення може мати кілька позицій |
| **Партія** (Batch) | Виробнича партія конкретної моделі. Створюється вручну з позиції замовлення |
| **Етап виробництва** (Stage) | Один із 6 кроків виробничого процесу: розкрій, пошив, оверлок, прямострочка, розпошив, упаковка |
| **Операція** (Operation) | Конкретна дія в межах етапу з формою для заповнення працівником |
| **Запис виконання** (Task Entry) | Фіктований результат роботи конкретного працівника на операції |
| **Маршрутна карта** (Route Card) | Технологічний маршрут для моделі продукту — послідовність операцій |
| **BOM** (Bill of Materials) | Специфікація матеріалів на одиницю продукту |
| **MRP** (Material Requirements Planning) | Розрахунок потреби матеріалів для замовлення |
| **Настил** (Nastil) | Шар тканини для розкрою. Містить параметри: ширина, довжина, вага, кількість |
| **Виробіток** | Кількість виконаних одиниць працівником за період |
| **Відрядна оплата** | Розрахунок зарплати на основі виробітку × ставка операції |

## Ролі користувачів

| Роль | Опис |
|------|------|
| `admin` | Повний доступ до всіх функцій системи |
| `manager` | Управління замовленнями, партіями, виробництвом |
| `master` | Підтвердження записів працівників, управління партіями |
| `technologist` | Створення маршрутих карт, моделей, операцій |
| `production_head` | Начальник виробництва, запуск партій |
| `quality` | Контроль якості, перегляд партій |
| `hr` | Управління персоналом, вакансії, навчання |
| `cutting` | Працівник розкрою |
| `sewing` | Працівник пошиву |
| `overlock` | Працівник оверлоку |
| `straight` | Працівник прямострочки |
| `coverlock` | Працівник розпошиву |
| `packaging` | Працівник упаковки |
| `worker` | Загальна роль працівника |
| `qc` | Контроль якості |

## Статуси замовлення

| Статус | Опис |
|--------|------|
| `draft` | Чернетка, ще не затверджено |
| `approved` | Затверджено, готове до запуску |
| `launched` | Запущено у виробництво |
| `in_production` | Виробництво триває |
| `completed` | Всі партії завершені |
| `warehouse_transferred` | Передано на склад |
| `closed` | Закрито (архів) |
| `cancelled` | Скасовано |

## Статуси партії

| Статус | Опис |
|--------|------|
| `created` | Створено, ще не запущено |
| `cutting` | На етапі розкрою |
| `sewing` | На етапі пошиву |
| `overlock` | На етапі оверлоку |
| `straight_stitch` | На етапі прямострочки |
| `coverlock` | На етапі розпошиву |
| `packaging` | На етапі упаковки |
| `ready` | Готова продукція |
| `shipped` | Відвантажено |
| `closed` | Закрито |
| `cancelled` | Скасовано |

## Статуси завдань

| Статус | Опис |
|--------|------|
| `pending` | Очікує прийняття |
| `accepted` | Прийнято працівником |
| `in_progress` | Виконується |
| `completed` | Завершено |
| `cancelled` | Скасовано |
