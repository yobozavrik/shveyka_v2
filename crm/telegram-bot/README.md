# Telegram Bot

AI ассистент для керування виробництвом через Telegram.

## Можливості

- 🎤 **Голосові повідомлення** — розпізнавання через Whisper API
- 💬 **Текстові команди** — повний доступ до AI ассистента
- 📎 **Файли** — Excel, PDF, CSV, зображення (OCR)
- 📦 **Виробництво** — замовлення, партії, запуск
- 👥 **HR** — працівники, зарплата, задачі
- 📊 **Аналітика** — дашборд, продуктивність
- 🔍 **Knowledge Base** — пошук в документах
- 📝 **Повний аудит** — логування кожної дії

## Встановлення

```bash
cd telegram-bot
pip install -r requirements.txt
cp .env.example .env
# Edit .env with your tokens
python -m bot.main
```

## Environment Variables

| Змінна | Опис |
|--------|------|
| `TELEGRAM_BOT_TOKEN` | Токен бота від @BotFather |
| `CRM_URL` | URL вашої CRM системи |
| `CRM_API_KEY` | API ключ для CRM |
| `OPENAI_API_KEY` | Ключ для Whisper API |
| `SUPABASE_URL` | URL Supabase проекту |
| `SUPABASE_KEY` | Anon ключ Supabase |
| `ALLOWED_TELEGRAM_IDS` | Дозволені Telegram ID |

## Команди

| Команда | Опис |
|---------|------|
| `/start` | Вітання |
| `/help` | Допомога |
| `/status` | Статус системи |
| `/me` | Ваш профіль |
| `/history` | Історія повідомлень |
| `/clear` | Очистити історію |

## Приклади

```
AI: скільки заробила Олена за квітень?
AI: запусти партію #123
AI: покажи топ працівників за тиждень
```

## Голосові повідомлення

Надішліть голосове повідомлення — воно буде:
1. Розпізнано через Whisper
2. Надіслано до AI ассистента
3. Відповідь надійде текстом

## Файли

Надішліть Excel/PDF/CSV файл з питанням — AI проаналізує вміст.
